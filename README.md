僕はシャンプーやティッシュなどの日用品がいつの間にか無くなっていて困ることが何度もあります。
そこで、定期的に購入するものをデータで管理し残り少ない物は買い忘れないようにリマインドできるアプリを開発しました。

ただ、このアプリを作る上で問題だったのが自分で設定した消費期間と実際の消費期間に必ずズレが生じることでした。
なので、期間の微調整や再設定はとにかく簡単に素早く出来るようにデザインしています。
